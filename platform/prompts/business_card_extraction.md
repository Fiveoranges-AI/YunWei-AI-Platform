# 核心原则：宁缺毋滥

只抽取**图片上你能清楚看见原文字符**的内容。

- 看不清就填 `null`，不要猜、不要补、不要翻译。
- 字符模糊、被遮挡、像水印或印章 → 填 `null`。
- 不要从邮箱域名、地址里的园区名、二维码说明、广告语、客户案例或合作品牌 logo 中"推断"公司名——那些都不是这个人的所属公司。
- 每个非 null 字段必须给 `source_excerpt`，**逐字**抄录名片上印的原文（≤ 30 字），不能改写、翻译、补全后缀；如果你写不出名片上对应的原文，这个字段就该是 `null`。

# 字段

- `name`: 姓名
- `title`: 职位/职务（"销售经理"、"总经理助理"等）
- `company_full_name`: 名片上印出来的公司/机构名称，**逐字抄录**。名片写"XX 有限公司/集团/股份/科技"等后缀就照抄，没写就别加。如果名片上只有英文名，就填英文名；只有品牌 logo 文字，就填 logo 文字。读不清就填 `null`。
- `company_short_name`: 名片上**独立**印出来的简称、品牌名或英文缩写。如果整张名片只有一个公司/品牌名，本字段填 `null`，不要复制 `company_full_name`。
- `phone`: 座机
- `mobile`: 手机号
- `email`: 邮箱
- `address`: 地址
- `wechat_id`: 微信号
- `website`: 网站

# 公司识别陷阱（避坑清单）

下列文字**不是**这个人所属的公司，看到时**不要**填进 `company_full_name` / `company_short_name`：

1. 二维码周围的"扫一扫加好友 / 关注我们"等说明文字。
2. 名片背面/页脚的广告语、slogan、口号。
3. 客户案例、合作伙伴、获奖品牌、行业协会的 logo。
4. 地址里出现的"XX 工业园 / XX 大厦 / XX 科技园 / XX 创业中心"等园区或楼宇名称。
5. 邮箱域名本身：`abc@xx.com` 里的 `xx.com` **不能**作为公司名。
6. 网址中的域名：除非该域名以中文/品牌名的形式作为 **logo 印在名片正面**，否则不能用域名当公司名。

# 多手机 / 多公司

- 多个手机号：最先出现的填 `mobile`，其余填 `phone`。
- 同一张名片印了中英文两个公司名：`company_full_name` 填中文，`company_short_name` 填英文。
- 集团 + 子公司同时出现：与姓名 / 职位 / 电话排版上最贴近的那个填 `company_full_name`。
